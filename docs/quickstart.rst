Quickstart
===========

.. toctree::
   :hidden:

   binary
   build

Pre-built binaries of |smelly| are available for both macOS and Linux. See the
:doc:`binary install instructions </binary>`. You can also :doc:`build from
source </build>`.

Additionally, you can use your favorite package manager to install the |smelly|
package, but note that some Linux distribution packages are woefully outdated.
|smelly| is available in a vast number of package repositories for macOS
and Linux.

.. image:: https://repology.org/badge/tiny-repos/smelly.svg
   :target: https://repology.org/project/smelly/versions
   :alt: Number of repositories smelly is available in

See :doc:`Configuring smelly <conf>` for help on configuring |smelly| and
:doc:`Invocation <invocation>` for the command line arguments |smelly| supports.

For a tour of smelly's design and features, see the :doc:`overview`.
